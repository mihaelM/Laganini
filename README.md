# Laganini
Faks projekt

Upute za prvo pokretanje:
1. Skinuti repozitorij
2. U direktoriju JelaSRostilja pokrenuti
$ start.py db init
$ start.py db migrate
$ start.py db upgrade
time se kreirala baza podataka
$ start.py runserver
pokrece se aplikacija te joj se pristupa preko browsera na adresi
http://localhost:5000
